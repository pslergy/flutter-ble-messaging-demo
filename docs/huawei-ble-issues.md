# Huawei BLE Connection Issues

Many Huawei devices have limitations when working with BLE.

Typical issues:

• device cannot act as Central reliably
• connection errors
• message transfer limitations

This demo implements a workaround by using
Huawei devices as BLE Peripheral (GATT server).

Another phone connects and retrieves messages.

See demo:
https://github.com/pslergy/flutter-ble-messaging-demo
